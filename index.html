How this works

Guide

Progressive overload

The goal every session is to do slightly more than last time — an extra rep, or the same reps with more weight. Each exercise has a rep range (like 8–10). Add reps each week until every set hits the top of the range, then add weight and drop back to the bottom of the range. This is the main driver of muscle and strength gains over time.

RIR — Reps in Reserve

RIR is how many more reps you could have done before failure. 2 RIR means you stopped with 2 good reps left in the tank. Most sets here target 1–2 RIR — hard, but not grinding, all-out sets. Isolation work is often pushed closer to 0–1 RIR since the injury risk is lower.

Deload weeks

Every 6th week, volume is cut roughly in half and target RIR jumps to 4–5 (noticeably easier). This lets accumulated fatigue dissipate so the next block starts fresh, rather than running until performance stalls or an injury forces a break.

The split

Push, Pull, and Legs each get trained once per week. Sessions run roughly 45–60 minutes: compound lifts first while you're freshest (lower reps, longer rest), isolation work after to finish off each muscle group (higher reps, shorter rest).

Logging a set

Enter the weight and reps you actually did, and how many reps you had left (RIR), then mark the set complete — the rest timer starts automatically. Your last performance on each exercise is shown for reference so you always know what you're trying to beat.
